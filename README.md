## How to set up
First clone the repository
```bash
git clone https://github.com/betich/341-classroom-notifier
cd 341-classroom-notifier
```

Then install dependencies
```bash
npm i
```

setup token file
token.json
```json
{
    "token": "<your token here>"
}
```

run bot
```bash
node index.js
```
